# Sudoku solver
Self explanitory, a program that generates a sudoku grid (easy or difficult) and offers to solve the sudoku. 

## Usage

```bash
python3 solver.py

Would you like to generate new board?
(y/n)
: y

Easy or Difficult puzzles?: (e or d)d

-------------------------

2 0 0  |  3 0 0  |  6 0 0 
9 0 7  |  4 0 0  |  3 5 0 
------------------------- 
0 2 0  |  0 0 9  |  0 0 0 
0 0 5  |  0 0 0  |  0 0 8 
0 0 0  |  7 0 0  |  0 1 0

(y/n):  y
```
